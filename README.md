All my dotfiles
===

Features
---
* git configuration
* zsh with powerlevel10k theme
* zoxide et fzf
* quality of life tools: batcat, exa

Installation
---
See [INSTALL.md](INSTALL.md)

Usage
---
```bash
git clone --depth=1 https://github.com/yscialom/dotfiles.git && cd dotfiles
./apply.sh
```

Modification
---
* Updates files and rerun `./apply.sh`.
* Commit your changes
